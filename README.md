# ProjektDrugiLetnik
Ta projekt(zanesljivogeslo.java) je preprost generator gesel, napisan v Javi.
Uporabnik vnese dolžino gesla, program pa nato generira naključno geslo iz:

-velikih črk

-malih črk

-številk

-simbolov

Kaj sem se naučil?
-kako uporabljati Scanner za branje vnosa uporabnika

-kako deluje Math.random() za generiranje naključnih vrednosti

-pretvarjanje številk v znake s (char)

-kako združiti več pogojev (if/else)

-strukturo preprostega Java programa

-kako naložiti projekt na GitHub in urediti README
